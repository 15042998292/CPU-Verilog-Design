# CPU-Verilog-Design
# 一种基于 Verilog 语言的 CPU 设计方法

## 项目简介
本项目为本科毕业设计，基于 Verilog 硬件描述语言完成一个简易 CPU 的设计与实现，
包括指令系统设计、数据通路构建及功能模块验证。

## 项目内容
CPU 整体架构设计
运算器设计
流水线设计
功能仿真与测试

## 开发环境
Verilog HDL

## 毕设进行过程中遇到的困难
（1）Verilog HDL 开发环境搭建
在 FPGA 设计过程中，需要先完成 Verilog HDL 的开发环境配置，包括工程创建、编译工具安装以及仿真环境部署等。如果软件版本不兼容，可能会导致工程文件无法正常打开、IP 核无法调用或者综合失败等问题。此外，不同 FPGA 开发板对应的器件型号也需要正确配置，否则会影响后续下载与运行。
（2）ISE/Vivado 工程配置问题
由于不同版本的 ISE 或 Vivado 对工程结构、约束文件以及 IP 核支持存在差异，因此在工程迁移或模块整合时，容易出现路径丢失、约束无效或模块识别失败等情况。同时，在使用 RAM IP 核时，还需要额外配置时钟、读写方式以及初始化参数，配置不当可能导致存储器工作异常。
